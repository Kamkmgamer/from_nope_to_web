# Product Requirements Document (PRD)

## 1. Product Vision

To become the premier resource for Arabic and English speakers to master modern full-stack development, bridging the gap between "tutorial hell" and professional engineering.

## 2. Target Audience

- Absolute beginners (students, career switchers).
- Junior developers looking to learn Next.js/T3 Stack.
- Arabic speakers lacking high-quality native technical content.

## 3. Key Features & Scope (MVP)

### Phase 1: The Foundation

- **Landing Page:** High-conversion design explaining the value prop.
- **Auth System:** Secure login.
- **Course Viewer:** Markdown/MDX based lesson rendering.
- **Basic Code Block:** Read-only syntax highlighting.
- **Localization:** Ar/En toggle working for static UI.

### Phase 2: Interactivity

- **Embedded Editor:** Write and run code.
- **Progress Tracking:** "Mark as Complete" and progress bars.
- **Quizzes:** Simple multiple-choice questions at end of modules.
- **Dashboard:** User stats and resume-learning shortcuts.

### Phase 3: AI & Advanced Modules

- **Cerebras Chatbot:** "Ask my Tutor" feature.
- **Backend Track:** Advanced tutorials on tRPC, Drizzle ORM, and PostgreSQL (T3 Stack).
- **Project Submissions:** Users can submit github links for review (auto or peer).

## 4. Design Guidelines

- **Aesthetic:** Clean, distraction-free, "Premium Educational" feel.
- **Themes:** Dark (Default for coding) and Light.
- **Typography:** Inter (En) and Noto Sans Arabic/IBM Plex Arabic (Ar).
