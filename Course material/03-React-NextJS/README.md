# Course 3: Modern React & Next.js

**Duration:** ~5 hours  
**Level:** Intermediate  
**Prerequisites:** JavaScript Mastery  
**Goal:** Build production-ready web applications with React

---

## Course Overview

React is the most popular library for building user interfaces. Companies like Facebook, Netflix, Airbnb, and countless others use React. This course teaches you to build modern, component-based applications with React and Next.js.

---

## What You'll Learn

By the end of this course, you will be able to:

- ✅ Build reusable UI components
- ✅ Manage application state with hooks
- ✅ Handle side effects (data fetching, timers)
- ✅ Build forms with validation
- ✅ Implement routing and navigation
- ✅ Style React applications
- ✅ Deploy to production with Next.js

---

## Lesson Breakdown

### Lesson 1: Components & Props (45 min)

_Building blocks of React_

- What is a component?
- JSX syntax and rules
- Function components
- Props: passing data to components
- Default props
- Children prop
- Component composition
- Conditional rendering
- Rendering lists with keys

**Project:** Build a product card grid

---

### Lesson 2: State with useState (45 min)

_Interactive components_

- Why state matters
- The useState hook
- Updating state
- State with arrays and objects
- Controlled components (forms)
- Lifting state up
- State management patterns

**Project:** Build a shopping cart

---

### Lesson 3: useEffect Hook (40 min)

_Side effects in components_

- What are side effects?
- The useEffect syntax
- Dependency array
- Running on mount
- Running on update
- Cleanup functions
- Fetching data
- Avoiding common pitfalls

**Project:** Build a data dashboard

---

### Lesson 4: Forms & Validation (40 min)

_Handling user input_

- Controlled vs uncontrolled inputs
- Handling form submission
- Validation strategies
- Error messages
- Multi-step forms
- Form libraries overview

**Project:** Registration form with validation

---

### Lesson 5: Context & State Management (45 min)

_Sharing state across components_

- The prop drilling problem
- Creating context
- Provider and Consumer
- useContext hook
- When to use context
- State management patterns
- Introduction to Redux (optional)

**Project:** Theme switcher and user authentication

---

### Lesson 6: Next.js Fundamentals (50 min)

_Production-ready React_

- What is Next.js?
- Pages and routing
- File-based routing
- Dynamic routes
- API routes
- Server-side rendering (SSR)
- Static site generation (SSG)
- Deployment with Vercel

**Project:** Build and deploy a blog

---

---

## How to Study

1. **Build along** - Type code yourself, don't copy
2. **Experiment** - Modify examples to see what happens
3. **Read errors** - React has helpful error messages
4. **Use DevTools** - React Developer Tools extension
5. **Build projects** - Apply concepts immediately
