# الدورة 1: أساسيات الويب - HTML & CSS

**المدة:** ~3 ساعات
**المستوى:** مبتدئ تماماً
**الهدف:** بناء مواقع ويب جميلة ومتجاوبة من الصفر

---

## نظرة عامة على الدورة

تأخذك هذه الدورة من الصفر (لا معرفة مسبقة) إلى بناء مواقع ويب كاملة بثقة. ستتعلم التقنيات الأساسية التي تشغل كل صفحة ويب على الإنترنت: HTML للهيكل و CSS للتنسيق.

---

## ماذا ستتعلم

بنهاية هذه الدورة، ستكون قادراً على:

- ✅ فهم كيف تعمل مواقع الويب (العميل/الخادم، HTTP، DNS)
- ✅ كتابة مستندات HTML صحيحة وذات معنى (Semantic)
- ✅ تنسيق المواقع باستخدام CSS (الألوان، الخطوط، المسافات)
- ✅ إنشاء تخطيطات مرنة باستخدام Flexbox
- ✅ بناء تصاميم متجاوبة تعمل على جميع الأجهزة
- ✅ نشر موقعك على الإنترنت

---

## تفاصيل الدروس

### الدرس 1: كيف يعمل الويب (15 دقيقة)

_فهم الأساس_

- نموذج العميل والخادم
- ماذا يحدث عندما تزور موقعاً
- لغات الويب الثلاث: HTML, CSS, JavaScript
- إنشاء أول ملف HTML خاص بك

**المفاهيم الأساسية:** العميل، الخادم، URL، DNS، HTTP

---

### الدرس 2: بناء جملة HTML والوسوم (30 دقيقة)

_بناء الهيكل_

- هيكل المستند (`<!DOCTYPE>`, `<html>`, `<head>`, `<body>`)
- عناصر النص (العناوين، الفقرات)
- الروابط والتنقل
- الصور والوسائط
- القوائم
- الجداول
- النماذج (Forms)
- HTML الدلالي (`<header>`, `<nav>`, `<main>`, `<footer>`)

**المشروع:** بناء صفحة ملف شخصي

---

### الدرس 3: محددات CSS (30 دقيقة)

_استهداف العناصر للتنسيق_

- طرق إضافة CSS الثلاث (inline, internal, external)
- محددات العنصر، الفئة (Class)، والمعرف (ID)
- دمج وتجميع المحددات
- قواعد الأولوية (Specificity)
- خصائص CSS الشائعة
- الفئات الزائفة (`:hover` وغيرها)

**المشروع:** تنسيق صفحة ملفك الشخصي

---

### الدرس 4: نموذج الصندوق في CSS (25 دقيقة)

_فهم المسافات والتخطيط_

- المحتوى، الحشو (padding)، الحدود (border)، والهامش (margin)
- Box-sizing
- أنواع العرض (block, inline)
- التعامل مع الطفح (Overflow)
- الظلال والزوايا الدائرية

**المشروع:** إنشاء بطاقات منسقة

---

### الدرس 5: تخطيط Flexbox (35 دقيقة)

_تخطيطات حديثة ومرنة_

- وعاء Flex والعناصر بداخله
- المحور الرئيسي والمحور المتقاطع
- Justify-content و Align-items
- Flex-grow, Flex-shrink
- خاصية الفجوة (gap)
- أنماط التخطيط الشائعة (شريط التنقل، البطاقات، التوسيط)

**المشروع:** بناء شبكة منتجات متجاوبة

---

## المتطلبات المسبقة
