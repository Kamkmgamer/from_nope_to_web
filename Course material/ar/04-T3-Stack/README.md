# الدورة 4: حزمة T3 Stack

**المدة:** ~8 ساعات
**المستوى:** متقدم
**المتطلبات المسبقة:** React & Next.js
**الهدف:** بناء تطبيقات احترافية، متكاملة، وآمنة برمجياً

---

## ما هي حزمة T3؟

حزمة **T3 Stack** هي طريقة شائعة لبناء تطبيقات الويب تركز على _البساطة، والنمطية، والأمان البرمجي الكامل_.

وهي ليست إطار عمل بحد ذاته، بل مزيج من أفضل الأدوات:

1. **TypeScript**: جافا سكريبت مع قواعد أمان.
2. **Tailwind CSS**: التنسيق بدون كتابة ملفات css.
3. **tRPC**: ربط الواجهة الأمامية والخلفية بدون مخطط API.
4. **Prisma / Drizzle**: التحدث إلى قاعدة البيانات الخاصة بك.
5. **NextAuth / BetterAuth**: التعامل مع تسجيل دخول المستخدمين.
6. **Next.js**: إطار العمل الذي يجمع كل شيء معاً.

---

## خطة الدروس

### المرحلة 1: شبكة الأمان

- **الدرس 1: أساسيات TypeScript** - لماذا تجعلك الأنواع أسرع.
- **الدرس 2: Tailwind CSS** - التنسيق بسرعة التفكير.

### المرحلة 2: الخلفية والبيانات

- **الدرس 3: قواعد البيانات مع Drizzle** - تخزين بياناتك بشكل دائم.
- **الدرس 4: المصادقة (Authentication)** - BetterAuth و NextAuth.

### المرحلة 3: الرابط

- **الدرس 5: tRPC** - استدعاء وظائف الخلفية من الواجهة الأمامية.

---

## لماذا هذه الحزمة؟

هل قمت يوماً بتغيير اسم قاعدة بيانات ونسيت تحديث كود الواجهة الأمامية؟ T3 يمنع هذا. إذا غيرت شيئاً في قاعدة البيانات، سيتحول كود الواجهة الأمامية إلى اللون الأحمر فوراً. هذا هو "أمان الأنواع من البداية للنهاية" (End-to-End Type Safety).
