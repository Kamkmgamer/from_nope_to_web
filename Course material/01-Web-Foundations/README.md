# Course 1: Web Foundations - HTML & CSS

**Duration:** ~3 hours  
**Level:** Absolute Beginner  
**Goal:** Build beautiful, responsive websites from scratch

---

## Course Overview

This course takes you from zero knowledge to confidently building complete websites. You'll learn the core technologies that power every webpage on the internet: HTML for structure and CSS for styling.

---

## What You'll Learn

By the end of this course, you will be able to:

- ✅ Understand how websites work (client/server, HTTP, DNS)
- ✅ Write valid, semantic HTML documents
- ✅ Style websites with CSS (colors, fonts, spacing)
- ✅ Create flexible layouts with Flexbox
- ✅ Build responsive designs that work on all devices
- ✅ Deploy your website to the internet

---

## Lesson Breakdown

### Lesson 1: How the Web Works (15 min)

_Understanding the foundation_

- The client-server model
- What happens when you visit a website
- The three languages of the web: HTML, CSS, JavaScript
- Creating your first HTML file

**Key Concepts:** Client, Server, URL, DNS, HTTP

---

### Lesson 2: HTML Syntax & Tags (30 min)

_Building the structure_

- Document structure (`<!DOCTYPE>`, `<html>`, `<head>`, `<body>`)
- Text elements (headings, paragraphs, emphasis)
- Links and navigation
- Images and media
- Lists (ordered and unordered)
- Tables for data
- Forms for user input
- Semantic HTML (`<header>`, `<nav>`, `<main>`, `<footer>`)

**Project:** Build a personal profile page

---

### Lesson 3: CSS Selectors (30 min)

_Targeting elements for styling_

- Three ways to add CSS (inline, internal, external)
- Element, class, and ID selectors
- Combining and grouping selectors
- Specificity rules
- Common CSS properties
- Pseudo-classes (`:hover`, `:focus`, `:first-child`)

**Project:** Style your profile page

---

### Lesson 4: The CSS Box Model (25 min)

_Understanding spacing and layout_

- Content, padding, border, and margin
- Box-sizing (content-box vs border-box)
- Display types (block, inline, inline-block)
- Overflow handling
- Box shadows and rounded corners

**Project:** Create styled cards

---

### Lesson 5: Flexbox Layout (35 min)

_Modern, flexible layouts_

- Flex container and flex items
- Main axis vs cross axis
- Justify-content and align-items
- Flex-grow, flex-shrink, flex-basis
- The gap property
- Common layout patterns (navbar, cards, centering)

**Project:** Build a responsive product grid

---

## Prerequisites

No prior experience required! Just bring:

- A computer (Windows, Mac, or Linux)
- A text editor (VS Code recommended)
- A web browser (Chrome recommended)
- Curiosity and patience

---

## How to Study

1. **Read each lesson carefully** - Don't rush
2. **Type out all code examples** - Don't just copy/paste
3. **Complete all exercises** - Practice is essential
4. **Build the projects** - Apply what you learn
5. **Experiment!** - Break things and fix them

---
